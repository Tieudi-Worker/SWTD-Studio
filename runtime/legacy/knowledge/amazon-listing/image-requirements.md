# Amazon Listing Image Requirements

## Source
- Amazon Seller Central official guidelines
- Category: amazon-listing
- Last verified: pending first research

## Main Image (Slot 1) Rules
- Pure white background (RGB 255,255,255)
- Product must fill 85% of frame
- No text, graphics, watermarks, borders
- No accessories not included in purchase
- Must be actual product photo (no illustrations)
- Minimum 1000px on longest side (recommended 2000px)
- JPEG, PNG, GIF, or TIFF format
- sRGB or CMYK color profile

## Secondary Images (Slots 2-8) Rules
- Can include text overlays, infographics, lifestyle scenes
- Minimum 1000px (recommended 2000px)
- Can show product in use, from different angles
- Can include comparison charts, size references
- Can include brand story elements

## A+ Content Image Specs
- Module widths: 970px standard, 1464px premium
- Heights vary by module type (600px, 300px, etc.)
- JPG or PNG, max 2MB per image
